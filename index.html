Network Troubleshooting Playbook

I created a structured troubleshooting guide for common home and campus network issues. It walks users through checking physical connections, verifying IP settings, and using tools like ping and traceroute.

View more technical details about this project

The playbook is organized as a description list for quick reference:

Layer 1
Check cables, Wi‑Fi signal strength, and power to devices.
Layer 2
Verify MAC addresses, switches, and basic connectivity.
Layer 3
Confirm IP addressing, gateways, and routing tables.

I also included a small script that parses logs and highlights repeated connection failures for faster diagnosis.
